## 🚀 Overview

A high-impact, animation-rich landing page inspired by the aesthetic and cinematic branding style of **Grand Theft Auto VI (GTA 6)**.

This project focuses on immersive UI design, smooth transitions, bold typography, and performance-optimized frontend engineering — built entirely using modern frontend technologies without frameworks.
